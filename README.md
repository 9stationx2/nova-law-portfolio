# ⚖️ Nova Law Portfolio - 제작중
> **법학의 논리를 데이터로 구조화하다**

노바법률사무소 송무팀 지원을 위해 제작된 리액트 기반 포트폴리오 사이트입니다. 
법률 전문성과 IT 기술력을 결합하여 법무행정의 디지털 전환 가능성을 제시합니다.

## ✨ Key Points
- **Legal Background**: 법학 전공으로 다져진 리걸 마인드
- **Data Expertise**: SQLD 자격 및 Oracle/MySQL 활용 능력
- **Systematic Approach**: 전산세무/회계 지식을 통한 증거 데이터 구조화
- **Continuous Growth**: 2026 정보처리기사 & 빅분기 취득 목표

## 📁 Project Structure
- `src/components/ui`: 재사용 가능한 UI 컴포넌트 (Button, Badge, Card)
- `src/sections`: 포트폴리오 각 섹션 (Intro, Expertise, Vision, Contact)
- `src/data`: 포트폴리오 문구 및 데이터 관리
